# TakyMed-Backend
Backend de l'application TakyMed

## Description
Partie Backend de l'application **TakyMed**, une solution intelligente de suivi de traitements médicaux assistée par IA. 
Ce service gère l'authentification, les posologies intelligentes, le stockage des ordonnances et la communication avec les pharmacies.

## Stack Technique
- **Langage :** Java 21 (OpenJDK)
- **Framework :** Spring Boot 4.0.1
- **Base de données :** PostgreSQL 18.1
- **Gestionnaire de dépendances :** Maven
- **Conteneurisation :** Docker

## Installation (Développement)
1. Cloner le dépôt : \`git clone https://github.com/votre-compte/TakyMed-Backend.git\`
2. Lancer la base de données : \`docker-compose up -d\` ou En local 
3. Compiler le projet : \`mvn clean install\`
4. Lancer l'application : \`mvn spring-boot:run\`
